# FIFA Predictions

Shareable FIFA match-winner tracker powered by Polymarket moneyline markets.

The published page is `index.html`. The updater refreshes Polymarket prices, grades resolved tracked picks, and rebuilds:

- `index.html`
- `fifa-predictions.html`
- `predictions.md`
- `data/predictions.json`

GitHub Actions is configured to refresh Tuesday, Thursday, and Saturday at 13:00 UTC.
